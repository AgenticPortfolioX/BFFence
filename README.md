# BFFence Company

Welcome to **BFFence**: A premier fence building company website crafted with passion by two best friends! We bring traditional pride and modern expertise to fencing in Southeast Michigan.

## Overview
This repository contains the source code for our modern, dynamic web presence. It highlights our specialized offerings, from custom wood fences to premium installations using PostMaster™ steel posts, and serves as our digital storefront for lead capture and quoting.

---

## 🚀 Run Locally

To get the site up and running in your local development environment:

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Setup Environment:**
   Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key (if leveraging AI Studio features).
3. **Start Development Server:**
   ```bash
   npm run dev
   ```

## 📈 SEO Optimization Checklist

To ensure maximum visibility and local search presence, follow this simple checklist to optimize the site for SEO:

- [ ] **Meta Titles & Descriptions:** Update `index.html` and any dynamic routing headers with localized, keyword-rich `<title>` and `<meta name="description">` tags (e.g., "Wood Fence Contractors in Southeast Michigan").
- [ ] **Semantic HTML:** Verify that every primary page contains exactly one `<h1>` tag with core keywords. Ensure headings cascade logically (`<h2>`, `<h3>`).
- [ ] **Image Alt Text:** Review all `<img>` tags to ensure they have descriptive `alt` attributes (like `alt="Custom Wood Privacy Fence Installation"`).
- [ ] **Sitemap & Robots.txt:** Generate an `sitemap.xml` and `robots.txt` file at the root to guide search engine crawlers.
- [ ] **Open Graph (OG) Tags:** Add OG and Twitter Card meta tags to `index.html` so links display beautifully (with images and descriptions) when shared on social media or in texts.
- [ ] **Local SEO Schema:** Embed JSON-LD Local Business Schema Markup in the HTML head to help Google interpret business hours, phone numbers, and location.
- [ ] **Performance (Core Web Vitals):** Ensure Vite creates a highly optimized build. Test loading speeds using Google PageSpeed Insights and compress any massive static images.
- [ ] **Google Business Profile:** Make sure the finalized domain is linked up with a verified Google Business listing for local map pack rankings.

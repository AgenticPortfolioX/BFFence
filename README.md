# BFFence Company

Welcome to **BFFence**: A premier fence building company website. We bring traditional pride and modern expertise to fencing in Southeast Michigan.

## Overview
This repository contains the source code for our modern, dynamic web presence. It highlights our specialized offerings, from custom wood fences to premium installations using PostMaster™ steel posts, and serves as our digital storefront for lead capture and quoting.

---

## 🚀 Run Locally

To get the site up and running in your local development environment:

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Setup Environment:**
   Configure your environment variables in `.env.local` based on `.env.example`.
3. **Start Development Server:**
   ```bash
   npm run dev
   ```

## 📈 SEO Optimization Checklist

To ensure maximum visibility and local search presence:

- [x] **Meta Titles & Descriptions:** Optimized `index.html` with localized, keyword-rich `<title>` and `<meta name="description">`.
- [ ] **Semantic HTML:** Verify that every primary page contains exactly one `<h1>` tag with core keywords.
- [x] **Image Alt Text:** All `<img>` tags have descriptive `alt` attributes.
- [ ] **Sitemap & Robots.txt:** Ensure proper `sitemap.xml` and `robots.txt` are in the root.
- [x] **Open Graph (OG) Tags:** Social media meta tags configured in `index.html`.
- [x] **Local SEO Schema:** JSON-LD Local Business Schema Markup embedded in the head.
- [ ] **Performance (Core Web Vitals):** Optimize build and compress static images.
